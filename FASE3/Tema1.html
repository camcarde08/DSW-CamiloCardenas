Sitio Web para el aprendizaje de la Interfaz de Usuario y la Interacción Humano Ordenado


El propósito de este sitio web, es brindar una herramienta didáctica presentando los diferentes temas que ayude a suplir las necesidades que se están presentando como resultado de la pandemia del Covid-19, adicionando las diferentes actividades de las fases



- Interfaz de Usuario


La interfaz es el punto en el que las personas y los ordenadores se ponen en contacto y se transmiten mutuamente tanto información,
órdenes y datos como sensaciones, intuiciones y nuevas formas de ver las cosas. En otras palabras, la interfaz de usuario de un
sistema contiene aquellos aspectos del sistema con los que el usuario entra en contacto, física, perceptiva o conceptualmente.


Importancia


Las personas no utilizan sistemas interactivos, sino que utilizan las interfaces que éstos les proporcionan; una parte muy importante
del éxito o fracaso de una aplicación depende de la interfaz dado a que es la parte con la que el usuario interactúan.
